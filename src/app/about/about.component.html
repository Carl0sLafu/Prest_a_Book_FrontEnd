Prest-a-Book


Esta web está hecha por Alberto Jiménez, Carlos Lafuente y Facundo Silva como parte del proyecto Prest-a-Book.

Prest-a-Book es una aplicación para el préstamos de libros que fomenta la lectura y el compañerismo entre personas con dicho hobby.

Este proyecto es posible gracias a T-System, Cambra Reus, Fundación esplai y Techma.

Prest-a-Book no se hace responsable de la pérdida de libros o diferencias entre los usuarios de esta plataforma.
